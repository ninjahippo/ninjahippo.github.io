the ultimate combination of robustness and versatility.

Ninjahippo is a CMS built for developers by developers. By combining the strengths of premier web technologies, a CMS may be created which is both fast, and convenient for use by developers.

why

Ninjahippo makes it incredibly easy for a developer to hand the management of a simple content website over to their client by giving them full access to the content management side, while maintaining control of the infrastructure management, design, etc.. Because Ninjahippo is completely API based, it is versatile enough to be used on a multitude of web platforms, all determined by the developer's discretion.

who

Ninjahippo is built and maintained by Joah Gerstenberg, but is opensourced for the contributions from the general public on GitHub.

how

The Ninjahippo CMS (cms.ninjahippo.io, available also on GitHub.) is built on Ruby on Rails, with Angular.JS serving up the frontend. The CMS is built as a graphical interface for the API (api.ninjahippo.io, available also on GitHub.), which is built on Node.JS and MongoDB, platforms known for their ability to serve blazing fast APIs.

get started

You can get started now by checking out the CMS or by reading through the GitHub docs repo.
